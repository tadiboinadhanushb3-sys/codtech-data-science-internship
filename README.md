CODTECH Data Science Internship Projects

This repository contains the projects completed during the CODTECH Data Science Internship.

Task 1 – Data Pipeline Development

Created a data preprocessing pipeline using Pandas and Scikit-learn. The pipeline loads data, cleans missing values, performs feature transformations, and saves processed datasets.

Task 2 – Deep Learning Model

Implemented an image classification model using TensorFlow and the MNIST dataset. The model was trained to recognize handwritten digits and achieved around 97% accuracy with visualization graphs for accuracy and loss.

Task 3 – End-to-End Data Science Project

Developed a house price prediction system including data preprocessing, model training, and deployment using a Flask API. The API accepts input features and returns predicted house prices.

Task 4 – Optimization Model

Solved a business optimization problem using Linear Programming with the PuLP library to maximize production profit under resource constraints.
